# Cross Tool Skill Creator

Use this skill when the user wants to create a new cross-tool skill source package in the `skills-repo` format.

## Default Output Order

1. skill name and summary
2. boundary and use cases
3. skill.yaml draft
4. core file drafts
5. validation scenarios
6. adapter generation next steps
