# Prompt 模板

## 模板 1：启动迭代开发

```text
请按“迭代式前端交付”执行本轮：
- 本轮目标：
- 用户动作：
- 验收信号：
约束：每轮改动保持小步，改完后必须跑 build gate。
```

## 模板 2：把口语反馈转为改动项

```text
下面是业务方反馈，请先改写成可执行改动项，再实施：
[粘贴反馈]
输出：
1) 可执行改动清单
2) 预计影响模块
3) 最小实现顺序
```

## 模板 3：每轮复盘

```text
请输出本轮迭代记录：
- Goal
- Delivered
- Feedback
- Gates（build/ux/behavior/boundary）
- Next
```
